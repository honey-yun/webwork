# 本人大三web期末的作业
使用mvc框架，个人博客网页
具体功能如下：
# （1）用户管理模块：

用户注册：新用户可以通过注册页面创建账号

用户登录：已注册用户可以通过用户名和密码登录系统

权限区分：系统区分管理员和普通用户两种角色

退出登录：用户可以安全退出系统

# （2）文章管理模块：

文章浏览：所有用户都可以浏览文章列表和详情

文章发布：管理员可以通过后台发布新文章

文章删除：管理员可以删除文章及相关评论（级联删除）

文章展示：首页展示最新5篇文章，列表页展示所有文章

# （3）评论系统模块：

评论查看：所有用户都可以查看文章评论

评论发布：登录用户可以发表评论

评论管理：管理员可以删除不当评论

# （4）后台管理模块：

管理首页：展示系统统计信息（如文章总数）

权限控制：确保只有管理员可以访问后台功能

响应式设计：后台界面适配不同设备屏幕
